# WellVantage Demo App Frontend


React Native mobile application for gym workout management.

## Features

- **Workout Management**: Create and manage custom workout plans
- **Availability**: Set personal trainer availability
- **Book Slots**: Book client slots for sessions

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Run on iOS:
```bash
npm run ios
```

4. Run on Android (requires Android SDK setup):
```bash
npm run android
```
   **Note:** Requires `ANDROID_HOME` environment variable to be set. If you get Android SDK errors, use `npm start` and scan QR code with Expo Go instead.

5. Run on Web:
```bash
npm run web
```
   - Opens in browser at `http://localhost:8081`

6. Build for Web Deployment:
```bash
npm run build:web
```
